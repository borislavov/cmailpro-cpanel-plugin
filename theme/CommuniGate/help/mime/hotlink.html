What is a Hotlink?

Also known as an “inline link,” a hotlink is a direct link that embeds a file (such as an image or video) from your site into another website. When another site embeds your files, it is using your bandwidth to serve those files.

What is Hotlink protection?

Hotlink protection prevents all sites that are not listed in the URLs to allow access field from accessing any extensions listed in the Block direct access for these extensions field.

Enable/disable Hotlink protection

  • Click the Enable or Disable button.

Note: Clicking the Disable button will delete URLs listed in the URLs to allow access field.

Allow access

1. Enter a URL in the URLs to allow access field.

2. Click Submit.

Block direct access for an extension

1. Enter the extension in the Block direct access for these extensions field.

2. Click Submit.

Note: Separate extensions with commas.

Allow users to access your content by entering a specific file’s URL into a browser

1. Click the Allow direct requests box.

2. Click Submit.

Redirect access request to a different URL

1. Enter the URL in the Redirect request to this URL box.

2. Click Submit.
