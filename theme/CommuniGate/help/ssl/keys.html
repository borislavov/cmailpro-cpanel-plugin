What is a private key (KEY)?

A private key is a string of characters that a computer uses to encode or decode encrypted messages it receives. It is integral to protecting your confidential information and should be safeguarded appropriately.

View or delete a key

  • To delete a key, click Remove from the Keys on Server table.
  • To view a key, click Show Details from the Keys on Server table.

Upload a new private key

If you have already generated a key, you can upload it using this screen.

1. Enter the key domain in the Domain the key is for box.

2. Paste the key in the Paste the key below box OR click the Choose File button.

3. Click Install.

Generate a new private key

If you haven’t generated a key yet, you can do so on this screen.

1. Enter the host in the Host box OR select the appropriate domain.

2. Select the appropriate key size.

3. Click Generate.
