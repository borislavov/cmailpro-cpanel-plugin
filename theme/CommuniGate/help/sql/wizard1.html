What is MySQL?

MySQL refers to a database management tool, as well as the type of database it manages. Such databases are used by applications like bulletin boards and retail shopping carts. The MySQL Wizard lets you create MySQL databases and user accounts.

To enter new information into a database, you can use the phpMyAdmin client installed with cPanel.

Create a new MySQL database

  1. Enter a database name in the New Database box.
  2. Click Next Step.
