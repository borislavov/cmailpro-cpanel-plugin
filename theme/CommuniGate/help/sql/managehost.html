What is MySQL?

MySQL refers to a database management tool, as well as the type of database it manages. Such databases are used by applications like bulletin boards and retail shopping carts.

What is the Remote MySQL feature?

Databases in cPanel can be accessed remotely, by other web servers. This can be useful if you wish to allow applications such as shopping carts or guestbooks on your other servers to access the databases.

This screen allows you to give external hosts (or servers) access to your MySQL databases.

Add an access host

  1. Enter an IP address or domain name in the Host box.
    Note: You may enter a fully qualified domain name (FQDN) or an IP address. You may also use % as a wildcard. For instance, to allow access from all IP addresses beginning with 208.77.188, you can enter 208.77.188.%.
  2. Click Add Host.

Delete an access host

  1. Click X. You will be directed to a new page.
  2. Click Yes to permanently remove the remote host's access, or click No to cancel the removal.
