What is PostgreSQL?

PostgreSQL refers to a database management tool, as well as the type of database it manages. Such databases are used by applications like bulletin boards and retail shopping carts. This screen lets you create, edit, and delete PostgreSQL databases and user accounts, and give users access to specific databases.

To enter new information into a database, you can use the phpPgAdmin client installed with cPanel.

Create a new PostgreSQL database

1. Enter a database name in the Database Name box.

2. Click the Create Database button.

Search for a PostgreSQL database

If you manage a large number of databases, you can use the Search field above the Current Databases table to locate a particular database. To search the Current Databases table, type your criteria into the Search field and click Go.

Delete a PostgreSQL database

1. Click the Delete Database link. You will be directed to a new page.

2. Click Yes to permanently remove the database, or click No cancel the removal.

Add a new PostgreSQL user

You will need to create PostgreSQL user accounts separately from mail and web administrator accounts.

1. Enter the username, password, and password confirmation in the appropriate boxes.

Note: The Strength box will show you the strength of your password choice. If you cannot think of a strong password, click the Password Generator button for an appropriate option.

2. Click the Create User button.

Add a PostgreSQL user to a database

For a user to be able to access a database, you will need to assign the user to the database.

1. Select the appropriate user from the User menu.

2. Select the appropriate database from the Database menu.

3. Click Submit.

Delete a PostgreSQL user

1. Click the X symbol. You will be directed to a new page.

2. Click Yes to permanently remove the user, or click No to cancel the removal.
