Create a new PostgreSQL user account

After you create the database, you will need to add a user so that the database can be filled and managed.

Please note that PostgreSQL user accounts must be created separately from mail and web administrator accounts.

  1. Enter the username and password in the appropriate boxes. Re-type the password in the Password (Again) box.
    Note: The Strength box will show you the strength of your password choice. If you cannot think of a strong password, click the Password Generator button for an appropriate option.
  2. Click Create User.
