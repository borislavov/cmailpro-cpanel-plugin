What is a filter?

A filter is a tool that processes mail according to your preferences. For example, a filter can automatically discard spam or save mail from a specified sender to its own folder.

In cPanel, filters can be applied to all email for the domain (Account Level Filters), or to each individual account (User Level Filters).

Create a new filter

  • Click the Create a New Filter button. You will be directed to a new page where you can choose features for your new filter.

View current filters

The Current Filters table allows you to view the names and actions of all current filters being applied to mail.

  • To edit a filter, click Edit.
  • To delete a filter, click Delete.

Conduct filter test

You can use the Filter Test feature to test how current filters will be applied to a sample message.

1. Enter a sample message in the box.
2. Click the Test Filter button. You will be directed to a page with the results.
