What is a filter?

A filter is a tool that processes mail according to your preferences. For example, a filter can automatically discard spam or save mail from a specified sender to its own folder.

In cPanel, filters can be applied to all email for the domain (Account Level Filters), or to each individual account (User Level Filters).

Create or edit a filter

1. Enter a name for your new filter in the Filter Name box.

  • Remember to use a different name for every filter. Using the same name will cause filters to be overwritten

2. Select the email characteristic that you want to use to filter mail.

3. Select the criterion (or operator) you want to use to filter mail.

4. In the text box, type the characters or keyword(s) you want to use to filter mail.

5. Click the + button to add the rule. You can add more than one rule.

  • Click the - button to delete a rule.

6. Select the action you want to take with mail “caught” by the filter.

7. Click the + button to add the action. You can add more than one action.

  • Click the - button to delete an action.

8. Click Activate.

Filter rules and actions

You can choose rules and actions from among several options. Our documentation includes a complete list of filter configuration options.

About operators

When you create a filter using several operators, it may be useful to note that cPanel will always process the “and” operator before the “or” operator. As a result, email filters will be processed as follows:

  • A or B and C will be processed as A or (B and C).
  • A and B or C will be processed as (A and B) or C.

If you wish to create a filter such as (A or B) and (C or D), you would have to create it like so:

  • (A and C) or (A and D) or (B and C) or (B and D).
