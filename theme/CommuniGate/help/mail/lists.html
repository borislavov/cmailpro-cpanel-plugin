What is a mailing list?

A mailing list is a group of email addresses that receive all mail sent to a single group address. You can easily send messages to several people, a useful feature for discussion lists or newsletter distribution.

Note: All mailing lists are handled by an application called Mailman. Please visit the Mailman site for more information.

Create a mailing list

1. Enter the list name in the appropriate field.

2. Click the Domain pull-down menu and select the appropriate domain.

3. Enter the password and password confirmation in the appropriate fields.

Note: The Strength box will show you the strength of your password choice. If you cannot think of a strong password, click Password Generator for an appropriate option.

4. Click Add Mailing List. A new page will appear.

Search mailing lists

1. Enter keyword(s) in Search field.

2. Click Go.

Delete a mailing list

  • Click Delete to the right of the corresponding list name. A new page will appear.

Change a mailing list password

  • Click Change Password to the right of the corresponding list name. A new page will appear.

Access Mailman options

  • Click Modify to the right of the corresponding list name. A new page will appear.
