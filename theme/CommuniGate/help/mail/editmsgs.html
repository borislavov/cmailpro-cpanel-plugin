Edit BoxTrapper verification or blacklist message

This screen allows you to edit the messages that BoxTrapper sends to your email correspondents.

The messages are:

  • verify — This message asks email senders to respond, verifying that they are not spam sources.
  • verifyreleased — This message confirms that the original sender has been whitelisted.
  • returnverify — BoxTrapper sends this message when it receives a verification email that is formatted incorrectly.
  • blacklist — This message informs the sender that your account will not accept their email.

To edit a message:

  • Click Edit to the right of the appropriate message name.

Reset BoxTrapper verification or blacklist message

  • Click Reset to default to the right of the appropriate message name.
