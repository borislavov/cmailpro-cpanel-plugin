What is Domain Lookup?

This feature allows you to look up data about a domain, such as the IP address or DNS zone information.

What is DNS?

DNS (Domain Name System) is the component of the Internet which acts as a “phone book,” converting human-readable domain names (such as www.example.com) into computer-readable IP addresses (such as 208.77.188.166).

Look up domain data

  1. Enter the domain in the Enter a domain to lookup box.
  2. Click Lookup.

What is Trace Route?

Trace Route displays the route your computer takes to access your website. This function will show you how many servers your data passes through before it reaches your website. Included in this information is the amount of time your computer took to reach the server.

Note: This function may be disabled; contact the system administrator for more information about using the Trace Route function.

Trace route from the server

  • Click Trace.
