What is PHP?

PHP is a computer scripting language in which many web-based applications are written. PHP applications are commonly found with the filename extensions .php, .php4, or .php5. Some PHP applications require PEAR packages, which can be installed in cPanel through the PHP PEAR Packages feature.

What is the PHP configuration?

This feature details PHP settings customized by the server administrator. The settings are for your reference only; you cannot edit them.

For detailed information about each directive, view our PHP Configuration documentation in the cPanel User Guide.

Select a PHP version

If your web host allows you to select the version of PHP that your website will use, you may select it from this screen.

  1. From the drop-down menu, select the version of PHP you wish to use, or select System Default to use the version your web host has chosen.
  2. Click Update.
