What is PHP?

PHP is a computer scripting language in which many web-based applications are written. PHP applications are commonly found with the filename extensions .php, .php4, or .php5. Some PHP applications require PEAR packages.

What are PHP PEAR packages?

A PHP PEAR package is a piece of software written in the PHP language.

What is PEAR?

PEAR (PHP Extension and Application Repository) is a repository of PHP code. cPanel allows you to search for and install PEAR packages consisting of PHP programs which can perform useful functions for your website.

What is Location of Your PHP Extensions and Applications?

The Path field shows the location on the server where any installed PEAR Packages will appear.

Find a PHP Extensions and Applications package

  • Enter keyword(s) in the Search box and click Go. You will be directed to a new page.

    • OR

  • Click Show Available PHP Extension(s) and Application(s). You will be directed to a new page.

Install a PHP Extensions and Applications Package

  1. Enter a package name in the Install a PHP Extensions and Applications Package box.
  2. Click Install Now.

Installed PHP Extension(s) and Application(s)

  • Click the appropriate button — Update, Reinstall, Uninstall, or Show Docs — to the right of the appropriate package name.

    • View system installed modules

    There are certain PEAR packages that come pre-installed in cPanel. You can see a list of these by clicking Show System Installed Modules. You will be directed to a new page.

    What is Perl?

    Perl is a language for web applications, known for its ability to process text. Perl applications are commonly found as .pl, .pm, and .cgi files and may require Perl modules. Perl modules can be installed within cPanel.

    What is a Perl module?

    A Perl module is a common piece of software that is reused often.

    What is CPAN?

    CPAN (the Comprehensive Perl Archive Network) is a repository of Perl code. cPanel allows you to search for and install Perl modules which can perform useful functions for your website.

    What is Location of Your Perl Modules?

    The Path field shows the location on the server where any installed Perl modules will appear.

    Find a Perl module

  • Enter keyword(s) in the Search box and click Go. You will be directed to a new page.

    • OR

  • Click Show Available Perl Module(s). You will be directed to a new page.

    • Install a Perl module

    1. Enter a name for the Perl module in the Install a Perl Module box.
    2. Click Install Now.

    Installed Perl module(s)

  • Click the appropriate button — Update, Reinstall, Uninstall, or Show Docs — to the right of the appropriate package name.

    • View system installed modules

    There are certain Perl modules that come pre-installed in cPanel. You can see a list of these by clicking Show System Installed Modules. You will be directed to a new page.
