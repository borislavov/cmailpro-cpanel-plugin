What is the Getting Started Wizard?

The Getting Started Wizard can help you learn more about web hosting. It can also help you set up features in cPanel.

What is Web Disk?

Web Disk is a feature of cPanel that lets you manipulate your web files by dragging and dropping, just as you do on your local computer.

Access Web Disk

1. Click the Access Web Disk button.

2. You will be directed to a new page. Choose your operating system by clicking on either the Apple, Microsoft, or Linux/BSD menu, and selecting the correct version.

3. We recommend that you click SSL to allow your computer to connect to the Web Disk securely.

4. Click Go.

5. Open the file and save it on your desktop.

Windows Vista® Users: If you are running Windows Vista on your desktop, you will need to download a patch from Microsoft’s website in order to use Web Disk.
