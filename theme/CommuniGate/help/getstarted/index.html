What is the Getting Started Wizard?

The Getting Started Wizard can help you learn more about web hosting. It can also help you set up basic features in cPanel.

Navigate the Getting Started Wizard

  • Click the pull-down menu and select the section you wish to view.

OR

  • Click Continue or Next Step.

Note: To wait and set up a particular feature at a later time, click Skip this step.

Select a language

By default, cPanel is set to display English.

  1. To change your language, click the menu and select your language.
  2. Click Change.
