What is the Getting Started Wizard?

The Getting Started Wizard can help you learn more about web hosting. It can also help you set up features in cPanel.

What is webmail?

Webmail allows you to check your email from any computer that has web access.

To check your mail, either click the Access cPanel Webmail link, or visit http://example.com:2096 or http://example.com/webmail (replacing example.com with your domain). Port 2096 is recommended, as it is a secure port, which protects your passwords and other information.

Create a shortcut

To create a shortcut to webmail, so that you can access it from your desktop or web browser, simply click the Access cPanel Webmail link and drag it to the location you want.
