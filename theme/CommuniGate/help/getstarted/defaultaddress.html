What is the Getting Started Wizard?

The Getting Started Wizard can help you learn more about web hosting. It can also help you set up features in cPanel.

What is a default address?

The email address to which cPanel routes any message sent to email accounts which do not exist at your domain (also called a “catch-all address”).

Options for unrouted email

You can redirect mail that is sent to an invalid email address at your domain. You have 5 options for mail that is sent to an address that does not exist.

  • To access the more advanced options, click on the Advanced Options link.

Discard with error to sender

1. Select Discard with error to sender (at SMTP time).

2. Enter the failure message in the Failure Message (seen by sender) box.

3. Click Change.

Forward to email address

1. Select Forward to email address.

2. Enter the appropriate email address in the Forward to email address box.

3. Click the Change button.

Forward to your system account

When your administrator created your cPanel account, a system email account was automatically created along with it. You can use your cPanel username and password to check the mail in this account. To have invalid email sent to this account:

1. Select Forward to your system account.

2. Click Change.

Discard

1. Select Discard (Not Recommended).

2. Click Change.

Pipe to a program

1. Select Pipe to a Program.

2. Enter the appropriate program path in the box next to the house icon.

3. Click Change.

Note: When piping to a program, you should enter a path relative to your home directory. If the script requires an interpreter such as perl or php, you should omit the “/usr/bin/perl” or “/usr/bin/php” portion. Make sure that your script is executable and has the appropriate hashbang at the top of the script. If you do not know how to add the hashbang, just make sure to name your script file with the correct extension and you will be prompted to have the hashbang added automatically.
