What is a backup?

A backup is a copy of your website’s files, directories, databases, and email configurations. Keeping a backup copy of your website on your personal computer is a wise precaution for many reasons, including possible data corruption and computer failure.

Run a full backup

A full backup will create an archive of your entire site. This includes all the files in your home directory, MySQL databases, email forwarders and filters. Simply click the Download or Generate a Full Web Site Backup button.

Download a system backup

If system backups have been enabled by your web host, you can download a compressed copy of all your website’s files here.

You will see options for downloading the most recent backup made at daily, weekly, or monthly intervals, depending on how often backups are made.

  • Click the button for the system backup you wish to download.

Run partial backups

  • Home directory – This backup will create an archive of all the files in your home directory, even files not included in your public_html directory. Click the Home Directory button to run.
  • MySQL database – You can archive MySQL databases you own. Click a link from the list to back up the corresponding database.
  • Email forwarders – You can archive email forwarders you have created. Click a link from the list to back up the corresponding domain’s forwarder.
  • Email filters – You can archive email filters you have created. Click the a link from the list to back up the corresponding domain’s filters.

What is a restore?

This feature allows you to restore parts of your site by uploading selected partial backup files.

Restore partial backups

1. Click the appropriate Browse button and select the backup file to restore.

2. Click the Upload button.
