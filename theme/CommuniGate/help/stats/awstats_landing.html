What is AWStats?

AWStats (Advanced Web Statistics) is a program that provides information about the visitors to your website in both graphical and statistical views. More information about AWStats can be found at the program’s site here.

View AWStats

  • Click the link to the right of the appropriate domain.

Key AWStats terms

Request — Each time a visitor asks to see a specific page or file on your site.

File — Any part of your site (an image, HTML page, PDF document, script, etc.).

Host — The individual computers that access your site. Multiple users on one computer will be treated as one host.

Data — The contents of the files in your site. Anytime someone views a file, data is transferred, which counts towards your bandwidth usage.

Referrer — A website that links or “refers” a visitor to your site.

Robot/spider — A robot, or spider, is a program that accesses a website in order to perform some action, such as index the site in a search engine or post automated forum messages.
