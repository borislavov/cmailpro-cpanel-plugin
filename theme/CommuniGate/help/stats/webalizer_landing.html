What is The Webalizer?

The Webalizer is a complex program that displays various statistics for your website using tables and graphs.

Full documentation for “The Webalizer” can be found at its creators’ website here.

View Webalizer statistics for a domain

1. Click the magnifying glass icon to right of the appropriate domain.
2. Click a month link to view that month’s statistics.

Key terms

Request — Each time a visitor asks to see a specific page or file on your site.

File — Any part of your site (an image, HTML page, PDF document, script, etc.).

Host — The individual computers that access your site. Multiple users on one computer will be treated as one host.

Referrer — A website that links or “refers” a visitor to your site.
