What is Analog?

Analog Statistics is a program that provides information about the visitors to your website in both graphical and statistical views. More information about Analog can be found at the program’s site here.

View Analog stats

1. Click the magnifying glass icon to the right of the appropriate domain name. You will be directed to a new page.

2. Click a month link to view that month’s Analog stats. You will be directed to a new page.

3. Click the Go To links to navigate.

Key Analog terms

Request — Each time a visitor asks to see a specific page or file on your site.

File — Any part of your site (an image, HTML page, PDF document, script, etc.).

Host — The individual computers that access your site. Multiple users on one computer will be treated as one host.

Data — The contents of the files in your site. Anytime someone views a file, data is transferred, which counts towards your bandwidth usage.

Referrer — A website that links or “refers” a visitor to your site.
