What is disk space usage?

Disk space usage is the amount of actual disk space being used by directories in your account. See the File Manager to view individual file sizes.

Note that a file or directory’s disk space usage is not the same as its size. Because of how computers store data, a file that has, for example, 20 bytes of data (i.e. its size is 20 bytes) might actually occupy 4,096 bytes on the disk (i.e. its usage would be 4,096 bytes). The difference between an individual file’s size and its disk usage is no more than a few thousand bytes, but a large number of files can make the total disk space usage substantially greater than the sum of individual file sizes.

Note also that this page’s directory tree and chart report only disk space usage in your account’s home directory (). There may be other parts of the system that contain substantial amounts of data for your account. See the Disk Space Usage figure in the main page’s Stats bar for your account’s total disk space usage.

Disk space usage view options

  • To sort the list of directories by name, select name.
  • To sort the list of directories by disk usage, select disk usage.
  • To show a directory’s subfolders, click the “+” symbol to its left.
  • To hide a directory’s subfolders, click the “-” symbol to its left.
