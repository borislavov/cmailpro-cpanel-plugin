What is an IP Address?

An internet protocol (IP) address is a number that identifies a computer on a network, making it possible for other computers to find and communicate with it.

What is the IP deny manager?

This screen can block a single IP address or a range of IP addresses from accessing your site.

Deny an IP address

1. Enter an IP address or domain name in the IP Address or Domain box.

2. Click Add.

IP address formats

  • Single IP Address
    example: 10.5.3.333
  • Range
    example: 10.5.3.3-10.5.3.40
  • Implied Range
    example: 10.5.3.3-40
  • CIDR (Classless Inter-Domain Routing)
    example: 10.5.3.3/32
  • Implied
    example: 10. (will block all IP addresses starting with “10”)

Remove a denied IP address

  • Click the link under Remove and to the right of the appropriate IP address.
