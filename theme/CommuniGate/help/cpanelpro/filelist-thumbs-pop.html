What is the Thumbnailer?

The Thumbnailer tool allows you to quickly change the resolution (size) of all images in a directory. Modifications occur on a percentage basis.

You can use this feature to create thumbnails of images for a gallery.

Navigate the Thumbnailer directory list

The current directory is always displayed at the top of the list.

  • Click a folder icon to open a subdirectory and view its contents.
  • Click Up One Level to open the directory that contains the one you're currently viewing.

Select a directory

Click the name of the directory whose images you wish to modify.
