What is the Convert Image Format feature?

There are many different types of image files. This feature allows you to change an image from one format to another.

Choosing an Image Format

Image format Best uses
jpg Complex images (photos, complex graphics, text, and colors).
gif Simple images (line art, small text, black and white images, icons, button images).
png Simple images (line art, small text, black and white images, icons, button images). These images handle transparency better than gifs do.
bmp When image quality is a concern and file size is not an issue.

Convert an image format

  1. Click the pull-down menu and select the appropriate format.
  2. Click Convert.
