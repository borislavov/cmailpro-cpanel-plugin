What is the Image Manager?

The Image Manager is a collection of tools that allow you to modify the images that you have uploaded to your site.

What is the Thumbnailer?

The Thumbnailer tool allows you to quickly change the resolution (size) of all images in a directory. Modifications occur on a percentage basis.

You can use this feature to create thumbnails of images for a gallery.

What is the Image Scaler?

The Image Scaler allows you to change the resolution (size) of a single image. You can choose to modify the image based on a pixel or percentage basis.

What is the Convert Image Format feature?

There are many different types of image files. This feature allows you to change an image from one format to another.

Navigate the Image Manager tools

Click the link for the tool you wish to use. You will be directed to a new page.
