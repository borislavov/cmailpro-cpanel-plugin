What is the Thumbnailer?

The Thumbnailer tool allows you to quickly change the resolution (size) of all images in a directory. Modifications occur on a percentage basis. (If you wish to scale images individually or based on number of pixels, you must use the Image Scaler tool.)

Change the resolution (size) of images

  1. Enter a width percentage in the first box and height percentage in the second box.
    Note: Deselect Keep Aspect Ratio if you wish to change the width and height percentages independently.
  2. Click Thumbnail Images to change the image sizes.
  3. Click Reset Dimensions to change image sizes back to the default.

After resizing your images, cPanel places them in a "thumbnails" subdirectory within the current folder. The thumbnail images will be created with the prefix "tn_".
