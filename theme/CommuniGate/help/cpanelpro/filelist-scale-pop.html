What is the Image Scaler?

The Image Scaler allows you to change the resolution (size) of a single image. You can choose to modify the image based on a pixel or percentage basis.

Navigate the Image Scaler directory list

The current directory is always displayed at the top of the list.

  • Click a folder icon to open a subdirectory and view its contents.
  • Click Up One Level to open the directory that contains the one you're currently viewing.

Select an image

Click the name of the image you wish to scale.

Note: Resizing an image multiple times may result in a loss of quality. We recommend retaining a copy of your old image when resizing, to ensure the image does not become distorted.
