What is the Image Scaler?

The Image Scaler allows you to change the resolution (size) of a single image. You can choose to modify the image on a pixel or percentage basis.

Change the resolution (size) of an image

  1. Enter a width (in pixels) and a height (in pixels) in the New Dimensions boxes, or enter a width percentage and height percentage in the Percent boxes.
    Note: Deselect Keep Aspect Ratio if you wish to change the width and height independently.
  2. Click the Retain a copy of the old image as box and type a filename to save a copy of the original image.
    Note: We recommend that you save a copy of the original image, because resizing an image may cause unexpected distortion.
  3. Click Scale Image.
  4. If you wish, you can click Reset Dimensions to change image sizes back to the defaults.
