Duplicate a branding style

1. Enter the new branding style’s name in the Name field.

  • This feature will duplicate the default branding style so that you may download or edit it without damaging the original.

2. Click Create.

3. You may now click 1 of 2 options to edit the style you just created:

  • Click Go Back to Editing (style) to return to the Edit Branding Style screen, where you can brand the theme with your own logos, headings, etc.
  • Click Go Back to Branding Editor to return to the Branding Editor home screen.

Upload a branding style

1. Click Browse.

2. Select the file from your computer.

  • The expected file type is a tarball (.tar.gz).

3. Click Upload.

  • Click Go Back to Editing (style) to return to the Edit Branding Style screen, where you can brand the theme with your own logos, headings, etc.
  • Click Go Back to Branding Editor to return to the Branding Editor home screen.
