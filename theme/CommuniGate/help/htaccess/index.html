What is a password protected directory?

This type of directory requires a viewer to enter a username and password before viewing its contents. You can let one or several users access a protected directory.

Navigate the directory list

The current directory is always displayed at the top of the list.

  • Click a folder icon to open a subdirectory and view its contents.
  • Click Up One Level to open the directory that contains the one you’re currently viewing.

Password protect a directory

  • Click the name of the directory you wish to password protect.
