What is a password protected directory?

This type of directory requires a viewer to enter a username and password before viewing its contents. You can let one or several users access a protected directory.

Set permissions

1. Click the Password protect this directory box.

2. Enter a directory name in the Name the protected directory box.

3. Click Save.

4. Enter a new user’s username, new password, and password confirmation in the appropriate boxes.

Note: The Strength box will show you the strength of your password choice. If you cannot think of a strong password, click the Password Generator button for an appropriate option.

5. Click Add/modify authorized user.

Note: You can add as many users as you wish. If you do not create any users to access the directory, no one will be able to view it online.

Delete a user

You can delete a user’s access privileges.

  • Select the user from the Authorized Users box and click Delete User.
