What is a leech?

A leech is a visitor who uses another person’s login information to access a restricted area of a website.

cPanel allows you to prevent leeching by redirecting likely offenders or disabling accounts whose passwords have been compromised.

Navigate the directory list

The current directory is always displayed at the top of the list.

  • Click a folder icon to open a subdirectory and view its contents.
  • Click Up One Level to open the directory that contains the one you’re currently viewing.

Select a directory

Click the name of the directory whose images you wish to convert.
