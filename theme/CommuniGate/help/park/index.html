What is a parked domain?

A parked domain is a second domain that points to your primary domain. When users attempt to access the parked domain, they will see your main site. For example, both http://www.cpanel.net and http://www.cpanel.com go to the same place, as cpanel.com is a parked domain for cpanel.net.

Note: In order to park a domain, you must register it with a valid domain name registrar and configure it to point to your DNS servers.

Create a parked domain

1. Enter the parked domain name in the Create a New Parked Domain box.

2. Click the Add Domain button.

Search parked domains for keywords

1. Enter the keywords in the Search box.

2. Click the Go button.

Open the parked domain’s public_html folder

  • To view and edit the parked domain’s files in the cPanel File Manager, click public_html under the Domain Root table heading.

Remove a parked domain

  • Click the Remove link to the right of the appropriate parked domain.
