What is FTP?

FTP (File Transfer Protocol) is a method of transferring files from one computer to another. cPanel comes equipped with FTP server software that allows you to connect to your website via FTP.

What is anonymous FTP?

Anonymous FTP is a process whereby visitors without FTP accounts may upload and download files to and from your site.

Anonymous FTP can be convenient if you wish to make files publicly available for downloading. However, it does present security risks.

When setting up anonymous FTP, it is important to protect any sensitive information by changing file permissions and directory access permissions.

Enable anonymous FTP access

1. Click the box next to Allow anonymous access to. This allows anyone to connect to that directory and download and modify files.

2. Click the box next to Allow anonymous uploads to. This allows anyone to connect to that directory and upload files.

Note: We do not recommend enabling anonymous uploads.

3. Click Save Settings.

Compose an anonymous FTP welcome message for visitors

1. Enter the message in the box.

2. Click Save Message.
