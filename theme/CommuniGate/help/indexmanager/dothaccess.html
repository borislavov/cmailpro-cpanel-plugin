Indexing options

  • Default System Setting - This is the default setting for all directories. Indexing will behave according to the server's configuration.
  • No Indexing - This will prevent visitors from seeing the directory’s contents. Instead, they will be directed to the index file for the directory. If no index file exists, visitors will see a 403 Forbidden error page.
  • Standard Indexing - The directory index appears in a text-only format. All files and folders will be shown by their names only.
  • Fancying Indexing - Information about the files, such as the size and time last modified, appears.

    • Configure Index Manager settings

    1. Click the button for the option you want.
    2. Click Save.
