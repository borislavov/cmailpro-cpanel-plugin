What is the Index Manager?

When accessing a directory (rather than a page) on your site, a visitor will typically see the index page for that directory. If no index page exists, the browser will display a list (or index) of the files in that directory.

The Index Manager allows you to control whether and how your site will display the list of files and folders within each directory.

Navigate the Index Manager

The current directory is always displayed at the top of the list.

  • Click a folder icon to open a subdirectory and view its contents.
  • Click Up One Level to open the directory that contains the one you're currently viewing.

Select a directory

Click the name of the directory whose indexing style you wish to change.
