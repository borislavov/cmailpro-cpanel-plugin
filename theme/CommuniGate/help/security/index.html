Security Policy

What is a security policy?

A security policy is a measure that a server owner takes to protect the server from attacks. cPanel now lets server owners protect the server by requiring cPanel users to answer security questions whenever they log into their accounts from an unfamiliar IP address.

What is the purpose of creating security questions and answers?

Answering security questions identifies you as the legitimate cPanel account owner when you log into your account from an unfamiliar IP address.

What are access IPs?

Access IPs are IP addresses from which you can access your cPanel account without having to answer security questions. For example, if you often use cPanel at a work computer and a home computer, you can specify the IP addresses of both, to prevent having to answer security questions when you log in.

You can see the most recent IP address from which you accessed cPanel by looking at the cPanel Home screen, under Stats, next to Last login from.

Update security questions and answers

To update your security questions or answers, click Update Security Questions and Answers.

Reset security questions and answers

To reset your security questions or answers, click Reset Security Questions and Answers.

Manage access IP addresses

To enter IP addresses from which you wish to access cPanel without answering security questions, click Manage Access IPs.
