Security Policy

What is the purpose of creating security questions and answers?

Answering security questions identifies you as the legitimate cPanel account owner when you log into your account from an unfamiliar IP address.

Update security questions and answers

  1. Select a security question using the menu next to the security questionâ€™s number, or enter a question in the Q: text field.

  3. After you define your security questions and answers, click continue to save them.
  4. Review your answers.
  5. Click Click here to change answers or Save Answers.
