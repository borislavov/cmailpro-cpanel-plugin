Manage Access IPs

What are access IPs?

Access IPs are IP addresses from which you can access your cPanel account without having to answer security questions. For example, if you often use cPanel at a work computer and a home computer, you can specify the IP addresses of both, to prevent having to answer security questions when you log in.

You can see the most recent IP address from which you accessed cPanel by looking at the cPanel Home screen, under Stats, next to Last login from.

Add an access IP address

To add an access IP:

  1. In the Access IP fields, enter the IP address you wish to add.
    • Note: You may use an asterisk (*) in the last octet. This asterisk is treated as a wildcard, meaning any IP address within the specified IP address block may skip security questions when logging in.
  2. When you finish entering the IP address, click Add.

Remove an access IP address

To remove an access IP:

  • Click Remove next to the IP address you wish to remove.
