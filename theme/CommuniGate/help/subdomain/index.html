What is a subdomain?

A subdomain is a section of your website that resides in a subdirectory of your home folder.

The subdomain’s URL is the same as your domain, but with an added prefix. For example, if your domain is example.com, then the subdomain’s URL will be subdomain.example.com.

Unlike a domain, you don’t need to register a subdomain.

Create a subdomain

1. Enter a subdomain prefix in the Subdomain box.

2. Select the appropriate domain.

3. Enter a home directory name for the subdomain in the Document Root box.

Note: This directory is where the subdomain’s files will be stored.

4. Click Create.

Search within existing subdomains

1. Enter the keyword(s) in the Search box.

2. Click Go.

Access the subdomain’s files

  • Click the link under Document Root in the Modify a Subdomain table. You will directed to the File Manager.

Manage redirection of a subdomain

  • Click the Manage Redirection link to the right of the subdomain. You will be directed to a new page.

Remove a subdomain

  • Click the Remove link to the right of the subdomain. You will be directed to a new page.
