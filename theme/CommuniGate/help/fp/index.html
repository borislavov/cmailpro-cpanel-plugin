What is FrontPage®?

FrontPage® is a Microsoft® application that allows you to edit your site in a WYSIWYG (what-you-see-is-what-you-get) format, rather than using HTML code.

What are FrontPage Extensions?

FrontPage Extensions allow you to publish your site directly from the application, allowing you to skip the FTP process.

Important: FrontPage extensions are no longer supported by Microsoft. We do not recommend using FrontPage extensions.

Additionally, installing or uninstalling FrontPage extensions will reset the password protect feature on all directories and will remove all .htaccess files, as well as clear any changes made to .htaccess files.
