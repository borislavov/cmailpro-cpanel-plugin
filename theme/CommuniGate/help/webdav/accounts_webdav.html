What is Web Disk?

Web Disk is a feature of cPanel that lets you manipulate your web files by dragging and dropping, just as you do on your computer.

Windows Vista Users: If you are running Windows Vista® on your desktop, you will need to download a patch from Microsoft’s website in order to use Web Disk. You can find the patch here.

Create a Web Disk account

1. Enter the new account’s login name, password, password confirmation, and directory to access in the appropriate boxes.

Note: The Strength box will show you the strength of your password choice. If you cannot think of a strong password, click Password Generator for an appropriate option.

2. Click Create.

Access the Web Disk account

1. In the Web Disk Account Management table, click the Access Web Disk button to the right of the Web Disk account you wish to access.

2. You will be directed to a new page. Choose your operating system by clicking on either Apple, Microsoft, or Linux/BSD and selecting your version.

3. We recommend that you click the SSL box to help secure your connection.

4. Click the Go button below your operating system.

5. Open the file and run it.

Change the Web Disk password

  • Click Change Password next to the appropriate account. A new window will appear.

Delete the Web Disk account

  • Click Delete next to the appropriate account. A new window will appear.

Password protect the Web Disk directory

You can password protect the Web Disk access directory, for additional security.

  • Click Password Protect next to the appropriate account. The Password Protect Directories screen will appear.
