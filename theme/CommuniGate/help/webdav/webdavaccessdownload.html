What is a Web Disk

A web disk is the ability to access your website's files from your local computer. It acts just like a local disk.

Setting up the Web Disk

You usually will only need one web disk to access your account. You can create other web disk logins with access to specific areas of your site in the Web Disk area of cPa nel. Most operating systems allow you to simply download our web disk installer script and run it to create a link to the web disk on your Desktop

Known Issues

Microsoft Windows Vista® is known to have problems connecting to web disks and may prevent the use of a web disk.
