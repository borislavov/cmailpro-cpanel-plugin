¿Qué es el FTP?

FTP (sigla que en inglés significa: File Transfer Protocol (protocolo de transferencia de archivos)), es un método para transferir archivos desde una computadora a otra. cPanel se presenta equipado con un software servidor de FTP que le permite conectarse su portal de Internet a través de FTP.

¿Qué es una sesión de FTP?

Una sesión de FTP en la duración de tiempo durante el cual un usuario se conecta a su portal mediante una conexión de FTP.

En caso de permitir el acceso FTP anónimo, se deben monitorear regularmente las sesiones de FTP en su servidor. Si detecta una sesión sospechosa, puede desconectarla.

  • Haga clic en Desconectar para terminar una sesión de FTP.
  • Cargue nuevamente la página en su navegador para refrescar la lista actual de sesiones de FTP.
