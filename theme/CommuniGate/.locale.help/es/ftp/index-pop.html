Ayuda de cuentas de FTP

¿Qué es el FTP?
FTP son las siglas en inglés que significan: File Transfer Protocol (protocolo de transferencia de archivos). Este método le permite transferir archivos hacia y desde el servidor en el que se encuentra su portal de Internet. Para transferir archivos, necesitará un cliente FTP y una cuenta FTP. La ventaja del FTP sobre descargar archivos desde un portal de Internet, es que con el FTP se pueden transferir directorios de archivos completos de una sola vez, en lugar de uno a uno.

¿Por qué utilizaría FTP?
Existen muchas razones para utilizar el FTP. A pesar de que no es tan seguro como otros métodos, tales como el SCP, posee algunas ventajas. Por ejemplo: Si usted tiene un depósito de múltiples archivos, o un directorio que muchas personas desean descargar, puede configurar el acceso FTP anónimo a dichos archivos, los cuales podrán ser descargados de una sola vez en vez de uno a uno. El FTP también puede ser utilizado para cargar múltiples archivos a su sitio, de una sola vez.

Control de acceso
Por favor advierta que el nuevo usuario tendrá acceso de lectura y escritura al directorio que elija y a todos los subdirectorios incluidos. Por ejemplo, si elige agregar al usuario john, y darle acceso al directorio /public_html/john, el usuario podrá agregar, editar, renombrar y eliminar cualquier archivo o directorio dentro de /public_html/john.

Si se ingresa una sola barra / en la casilla de directorio, el nuevo usuario de FTP tendrá acceso a la totalidad del directorio public_html, y todos los subdirectorios incluidos en él. Si desea que el nuevo usuario tenga acceso solamente a un sub dominio, simplemente configure la casilla del directorio con el mismo valor del nombre de raíz de subdominio, que se enumera entre paréntesis en la sección de subdominios de este panel de control.

Cuentas para anotar
Las cuentas anónimas son utilizadas para el acceso FTP anónimo y no pueden ser eliminadas. Las cuentas de acceso a registros son utilizadas para descargar sus archivos de registro del sitio, y no pueden ser eliminadas. Su cuenta principal de FTP tiene acceso a todos los archivos dentro de su dominio, y tampoco puede ser eliminada.

¿Qué es el FTP anónimo?
El FTP anónimo le permite a cualquier persona ingresar a un directorio específico sin utilizar nombre de usuario o contraseña. Esto significa que cualquiera puede ingresar a los archivos que se encuentran en su directorio anónimo. Su directorio anónimo es public_ftp/.
