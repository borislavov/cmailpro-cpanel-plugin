¿Qué es PHP?"

PHP es un lenguaje de programación en el cual se escriben muchas aplicaciones basadas en Internet. Las aplicaciones de PHP se encuentran a menudo con las extensiones de archivo .php, .php4, o .php5. Algunas aplicaciones de PHP requieren paquetes de PEAR, que pueden ser instalados en cPanel mediante la característica de Paquetes PEAR de PHP.

¿Qué es la configuración de PHP?

Esta característica detalla las configuraciones de PHP personalizadas por el administrador del servidor. Las configuraciones son solamente como referencia, no se pueden editar.

Para obtener información detallada acerca de cada instrucción, vea nuestra documentación Configuración de PHP en la Guía del usuario de cPanel.

Seleccione una versión de PHP

Si la empresa que hospeda su portal de Internet le permite seleccionar la versión de PHP que su sitio utilizará, puede seleccionarla a partir de esta pantalla.

  1. En el menú desplegable, seleccione la versión de PHP que desea utilizar, o seleccione Valores predeterminados del sistema para utilizar la versión elegida por la empresa que hospeda su portal de Internet.
  2. Haga clic en Actualizar.
