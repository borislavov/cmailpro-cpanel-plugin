¿Qué es un tipo MIME?

Los tipos MIME indican a los navegadores cómo tratar extensiones específicas. Por ejemplo, el tipo MIME text/html equivale a .htm, .html y .shtml en la mayoría de los servidores, lo que indica al navegador que debe interpretar todos los archivos con estas extensiones como archivos HTML. Se pueden alterar o agregar nuevos tipos MIME específicamente para su sitio (nota: No se pueden alterar los valores de tipos MIME definidos por el sistema). Los tipos MIME se usan a menudo para gestionar nuevas tecnologías cuando estas aparecen.

Un tipo MIME es un tipo de medio de Internet que se envía a una aplicación cliente, para que pueda gestionar el archivo apropiadamente. Básicamente, garantiza que el navegador de red entiende qué hacer con los archivos que le han sido enviados.

Por ejemplo: Si desea alojar archivos .wav y hacer que los visitantes de su sitio sean capaces de accederlos, puede agregar el tipo mime audio/x-wav para los archivos .wav.

Modificar un tipo MIME le permite cambiar la forma en la que un tipo de archivo se presenta al navegador de red de un visitante. Por ejemplo, tal vez desee tener todos los archivos .music presentados como archivos tipo wave. Para ello, debe agregar el tipo mime audio/x-wav para los archivos .music.

 

Agregar tipos MIME

Para agregar un tipo MIME, necesitará especificar el tipo de archivo en el campo Tipo MIME, y la extensión o extensiones de archivo a ser asociadas con este tipo Mime en el campo Extensión(es), luego haga clic en Agregar

 

Remover tipos MIME

Cualquier tipo MIME agregado por el usuario aparecerá en verde. Esos pueden ser eliminados. Cualquier otro tipo MIME son tipos de sistema y no pueden ser eliminados.

Para eliminar un tipo MIME, asegúrese de que la caja de selección próxima se encuentra seleccionada, y seleccione Eliminar seleccionado de la lista desplegable Acciones....
