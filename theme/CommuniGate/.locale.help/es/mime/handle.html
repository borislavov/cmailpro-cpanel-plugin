¿Qué es Apache?"

Apache es un programa que recibe solicitudes provenientes de navegadores de red. Responde "sirviendo" portales de Internet a los navegadores; por esta razón, se denomina software servidor de la red.

¿Qué es un gestor de Apache?

Los gestores Apache controlan cómo el programa de servidor de Internet Apache administra ciertos tipos de archivos y extensiones. Apache está preconfigurado para tratar a scripts CGI y archivos procesados por servidor. Sus extensiones de archivo incluyen: .cgi, .pl, .plx, .ppl, .perl, y .shtml.

El sistema Apache se puede configurar para manejar un nuevo tipo de archivo con un gestor existente, agregando manualmente el gestor y la extensión en esta pantalla. Por ejemplo, para hacer que el servidor gestione un archivo con la extensión .ejemplo como archivos CGI, se debe ingresar ”cgi-script” debajo de Gestor y “.ejemplo” debajo de Extensión(es).

Gestores disponibles

Los siguientes gestores de apache están disponibles. Ellos son seguidos de un nombre de módulo entre paréntesis. Por favor advierta que el módulo entre paréntesis debe estar instalado dentro de Apache en el servidor:

default-handler: Envía el archivo utilizando el gestor predeterminado de apache de contenido estático. (core)
send-as-is: Envía el archivo con encabezados HTTP como están (mod_asis)
cgi-script: Maneja el archivo como una secuencia CGI. (mod_cgi)
imap-file: Procesa el archivo como un archivo de regla imagemap; para más información vea la documentación de Apache. (mod_imap)
server-info: Recupera la información de configuración del servidor. (mod_info)
server-parsed: Realiza el procesamiento de archivos de archivos incluidos del lado servidor. (mod_include)
server-status: Recupera la información de estado del servidor. (mod_status)
type-map: Procesa el archivo como un archivo tipo mapa; para más información vea la documentación de Apache. (mod_negotiation)

Aviso: Esta interfaz no le permite crear gestores de Apache personalizados.

Crear un gestor de Apache

  1. Ingrese la ruta de archivo de la acción/aplicación en la casilla Gestor.
  2. Ingrese las extensiones en la casilla Extensiones. Se pueden utilizar espacios para separar extensiones múltiples.
  3. Haga clic en Agregar.

Remover gestores Apache definidos por el usuario

Se pueden remover gestores Apaches definidos por el usuario.

Aviso: Las extensiones de archivo gestionadas automáticamente por Apache no pueden ser removidas.

  1. Haga clic en X a la derecha del gestor apropiado en el gráfico Gestores de Apache definidos por el usuario. Será dirigido a una nueva página.
  2. Haga clic en Sí para remover el gestor o en No para cancelar la remoción.
