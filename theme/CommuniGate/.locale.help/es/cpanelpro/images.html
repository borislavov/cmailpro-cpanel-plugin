¿Qué es el Administrador de imágenes?

El Administrador de imágenes es una colección de herramientas que le permite modificar las imágenes cargadas en su portal.

¿Qué es el Miniaturizador?

La herramienta Miniaturizador le permite cambiar rápidamente la resolución (tamaño) de todas las imágenes en un directorio. Las modificaciones ocurren sobre la base de un porcentaje.

Se puede utilizar esta característica para crear miniaturas de imágenes de una galería.

¿Qué es el Escalador de imágenes?

El Escalador de imágenes le permite cambiar la resolución (tamaño) de una imagen individual. Se puede elegir modificar la imagen, basado en píxeles o en porcentajes.

¿Que es la característica Convertir el formato de imagen?

Existen muchos tipos diferentes de archivos de imagen. Esta característica le permite cambiar una imagen de un formato a otro.

Navegar las herramientas del Administrador de imágenes

Haga clic en el vínculo de la herramienta que desea usar. Será dirigido a una nueva página.
