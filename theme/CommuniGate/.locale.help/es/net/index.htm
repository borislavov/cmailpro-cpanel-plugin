¿Qué es la Búsqueda de información de dominios?

Esta característica le permite obtener información acerca de un dominio, como por ejemplo datos como la dirección IP o información de zona de DNS.

¿Qué es el DNS?

DNS (en inglés: Domain Name System, (sistema de nombres de dominio)) es el componente de Internet que actúa como una especie de “agenda telefónica”, convirtiendo los nombres de dominio legibles para los seres humanos (como: www.ejemplo.com) en direcciones IP legibles para la computadoras (como: 208.77.188.166).

Buscar información de datos de dominio

  1. Ingrese el dominio de la casilla Ingresar un dominio para buscar información.
  2. Haga clic en Buscar.

¿Qué es Traceroute (trazar ruta)?

Traceroute (trazar ruta) es la aplicación que muestra la ruta que su computadora recorre para ingresar a su portal de Internet. Esta función le mostrará por cuantos servidores viajan sus datos antes de llegar a su portal de Internet. En esta información se encuentra incluido el tiempo requerido por su computadora para llegar al servidor.

Aviso: Esta función puede ser desactivada; contacte al administrador del sistema para obtener más información acerca del uso de la función Traceroute.

Ejecutar la función Traceroute desde el servidor

  • Haga clic en Rastrear.
