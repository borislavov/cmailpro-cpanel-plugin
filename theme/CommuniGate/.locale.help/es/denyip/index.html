¿Que es una dirección IP?

Una dirección de protocolo de Internet (IP) es un número que identifica a una computadora en una red, haciendo posible que otras computadoras la encuentren y se comuniquen con ella.

¿Que es un administrador de rechazo de IP?

Esta pantalla puede bloquear a una única dirección IP o rango de direcciones IP, evitando el ingreso a su sitio.

Rechazar una dirección IP

1. Ingrese una dirección IP o nombre de dominio en la casilla Dirección IP o dominio.

2. Haga clic en Agregar.

Formatos de dirección IP

  • Dirección IP individual
    ejemplo: 10.5.3.333
  • Rango
    ejemplo: 10.5.3.3-10.5.3.40
  • Rango implícito
    ejemplo: 10.5.3.3-40
  • CIDR (Ruteo inter dominios sin clases)
    ejemplo: 10.5.3.3/32
  • Implícito
    ejemplo: 10. (bloqueará todas las direcciones IP que comiencen con "10")
