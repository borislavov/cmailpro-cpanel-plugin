Eliminar el bloqueo de una dirección o rango de direcciones IP

Esta pantalla le permite confirmar su voluntad de remover una dirección o rango de direcciones IP de la lista de direcciones bloqueadas para ingresar a su sitio.

  • Haga clic en Eliminar IP para permitir que las direcciones IP ingresen a su sitio.
  • Haga clic en cancelar para volver a la pantalla principal de Administrador de rechazo de IP.
