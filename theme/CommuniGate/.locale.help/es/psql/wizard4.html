Completar la tarea

CPanel muestra un mensaje afirmando que la base de datos y la cuenta de usuario fueron instaladas con éxito.

A partir de aquí, puede seleccionar una de tres opciones:

  • Agregar otra base de datos — Vuelve al inicio del Asistente de bases de datos PostgreSQL.
  • Agregar otro usuario a su base de datos PostgreSQL — Abre la página Bases de datos , desde donde se pueden crear cuentas de usuarios adicionales y asignarlos a la base de datos.
  • Volver al inicio — Abre la pantalla inicial de cPanel.
