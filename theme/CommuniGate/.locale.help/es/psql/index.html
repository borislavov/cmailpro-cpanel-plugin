¿Qué es PostgreSQL?"

PostgreSQL se refiere a una herramienta de administración de base de datos, así como también al tipo de base de datos que administra. Dichas bases de datos son utilizadas por aplicaciones tales como boletines de noticias y carros de compras de venta al por menor. Esta pantalla le permite crear, editar y eliminar bases de datos y cuentas de usuario de PostgreSQL, y le da a los usuarios acceso a bases de datos específicas.

Para ingresar nueva información en una base de datos, se puede utilizar el cliente phpPgAdmin instalado con cPanel.

Crear una nueva base de datos de PostgreSQL

1. Ingrese un nombre de base de datos en la casilla Nombre de base de datos.

2. Haga clic en el botón Crear bases de datos.

Buscar una base de datos de PostgreSQL

Si se administra una cantidad importante de bases de datos, se puede utilizar el campo Buscar arriba de la tabla Bases de datos actuales para localizar una base de datos en particular. Para buscar en la tabla Bases de datos actuales ingrese sus criterios en el campo Buscar y haga clic en Ir.

Eliminar una base de datos de PostgreSQL

1. Haga clic en el vínculo Eliminar bases de datos. Será dirigido a una nueva página.

2. Haga clic en Sí para eliminar permanentemente la base de datos o en No para cancelar la remoción.

Agregar un nuevo usuario de PostgreSQL

Necesitará crear cuentas de usuarios de PostgreSQL en forma separada a las cuentas de correo y administrador de red.

1. Ingrese el nombre de usuario, contraseña y confirmación de la contraseña en las casillas apropiadas.

Aviso: La casilla Seguridad le mostrará la seguridad de su contraseña. Si no puede imaginarse una contraseña segura, haga clic en el botón Generador de contraseñas para obtener una opción apropiada.

2. Haga clic en el botón Crear usuario.

Agregar un usuario de PostgreSQL a la base de datos

Para que un usuario pueda ingresar a la base de datos, necesitará asignarlo a dicha base.

1. Seleccione el usuario apropiado del menú Usuario.

2. Seleccione la base de datos apropiada del menú Base de datos.

3. Haga clic en Enviar.

Eliminar un usuario de PostgreSQL

1. Haga clic en el símbolo X. Será dirigido a una nueva página.

2. Haga clic en Sí para remover permanentemente el usuario o en No para cancelar la remoción.
