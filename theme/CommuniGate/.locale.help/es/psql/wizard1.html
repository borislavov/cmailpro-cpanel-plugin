¿Qué es PostgreSQL?

PostgreSQL se refiere a una herramienta de administración de base de datos, así como también al tipo de base de datos que administra. Dichas bases de datos son utilizadas por aplicaciones tales como boletines de noticias y carros de compras de venta al por menor.

Para ingresar nueva información en una base de datos, se puede utilizar el cliente phpMyAdmin instalado con cPanel.

Crear una nueva base de datos de PostgreSQL

1. Ingrese un nombre de base de datos en la casilla Nueva base de datos. .

2. Haga clic en el botón Siguiente paso.
