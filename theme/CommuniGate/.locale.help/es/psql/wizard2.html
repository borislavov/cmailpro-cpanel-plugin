Crear una nueva cuenta de usuario de PostgreSQL

Después de crear la base de datos, necesitará agregar un usuario para que dicha base de datos pueda ser completada y administrada..

Por favor advierta que las cuentas de usuarios de PostgreSQL deben crerase en forma separada de las cuentas de correo y administrador de red.

  1. Ingrese el nombre de usuario y contraseña en las casillas apropiadas. Ingrese nuevamente la contraseña en la casilla Contraseña (nuevamente).
    Nota: La casilla Seguridad le mostrara la seguridad de su contraseña. Si no puede imaginarse una contraseña segura, haga clic en el botón Generador de contraseñas para obtener una opción apropiada.
  2. Haga clic en Crear usuario.
