Cuadro de relaciones de PostgreSQL

El siguiente cuadro está diseñado para mostrar las relaciones entre los usuarios de PostgreSQL y las bases de datos:

En este ejemplo, los usuarios Ay B. tienen la capacidad de escribir en la base de datos Database1 También, el usuario B puede leer y escribir en la base de datos Database2. El propósito de este cuadro es mostrar que las bases de datos y los usuarios de PostgreSQL, son objetos independientes y se necesitan crear conexiones entre ellos para que puedan interactuar.
