Editor de zona DNS avanzado

El Advanced Zone Editor (editor de zona DNS avanzado) le permite un mayor control en la configuración de registros de zona de DNS. Aquí se puede configurar su archivo de zona con las especificaciones que desee, tales como TTL, dirección IP, y el tipo de registro (A, CNAME, o TXT)

¿Qué es TTL?

TTL significa "tiempo de vida", y es un límite al período de tiempo o número de transmisiones, que una unidad de datos puede experimentar antes de ser descartada. En el DNS ocurren TTLs, cuando estos son configurados por un servidor de nombres autorizado, para un registro particular de recursos. Los TTL se pueden configurar en segundos (por ejemplo: 14.400) para describir el momento en el que desea que el registro sea descartado.

Tipos de registros

Registro A: Un registro A, o registro de direcciones, hace coincidir su nombre de dominio con una dirección IP.

Registro CNAME: Un registro CNAME, o registro de nombre canónico, hace coincidir su nombre de dominio con un alias. Esto es útil si está ejecutando un servicio diferente al del servidor en la red, tal como servidor FTP, en un puerto separado.

Registro MX: Un registro MX, o registro de agente de transporte de correo, le dice a un programa cliente cuál servidor recibe el correo enviado a un nombre de dominio.
