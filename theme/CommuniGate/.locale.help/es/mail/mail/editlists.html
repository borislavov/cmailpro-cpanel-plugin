Editar la lista de BoxTrapper

Aquí se puede seleccionar una lista de BoxTrapper que desee modificar. Las listas de BoxTrapper contienen las direcciones de correo electrónico de las personas a quienes les envía correo.

  • Haga clic en el vínculo que corresponde a la lista que desea modificar.

Las listas disponibles son:

  • Lista negra — Una lista de direcciones de correo electrónico, cuyos mensajes entrantes serán automáticamente bloqueados. BoxTrapper envía una notificación a los remitentes que se encuentran en esta lista.
  • Lista ignorar — Una lista de direcciones de correo electrónico, cuyos mensajes entrantes serán bloqueados. BoxTrapper no envía mensajes de advertencia luego de recibir un correo electrónico proveniente de una dirección ignorada.
  • Lista blanca — Una lista de direcciones de correo electrónico, cuyos mensajes entrantes serán automáticamente aceptados.
