¿Qué es el SSH?

El SSH (intérprete seguro) es un protocolo de red que le permite ingresar a archivos en su servidor de la red a través de Internet utilizando una interfaz de línea de comandos (CLI).

Importante: SSH le da acceso directo a los archivos en el servidor. Tenga precaución para no modificar archivos de una forma tal que pudieran causar problemas en su servidor.

Acerca de la codificación de clave pública

La codificación es muy importante para mantener privadas las comunicaciones de SSH. La codificación de la clave pública es un método de comunicación que utiliza un par de claves coincidentes (una pública, y una privada), para crear un código secreto. Esto garantiza que cualquiera que tenga acceso a los mensajes enviados hacia o recibidos por su computadora, no pueda leerlos, sin acceso a la clave privada.

Una clave privada es una secuencia de caracteres que una computadora usa para codificar o decodificar mensajes codificados. Mantener esta clave en forma secreta es esencial para proteger su información confidencial.

Una clave pública es una secuencia de caracteres que una computadora usa para codificar o decodificar mensajes que recibe. Típicamente una clave pública será colocada en un servidor, para de esa forma acceder al servidor con codificación.

Administrar claves de SSH

  • Para crear, importar, y administrar claves de SSH, haga clic en Administrar claves de SSH.

¿Que son SSH Term y el applet Java Telnet?

Estas opciones le permiten conectarse al servidor utilizando SSH, sin descargar ningún software, utilizando una clave o autenticación de contraseña. Estas acciones requieren de Java.

Iniciar la sesión utilizando SSH Term

Para utilizar SSH Term, debe tener una clave privada instalada en su computadora. Para instalar una, haga clic en Administrar claves de SSH.

  • Haga clic en Conectar utilizando SSH Term.

Inicia la sesión utilizando el applet Java Telnet

El applet Java Telnet requiere que utilice una contraseña para iniciar la sesión en su servidor.

  • Haga clic en Conectar utilizando el subprograma Java Telnet.
