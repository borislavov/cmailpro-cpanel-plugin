¿Qué es MySQL?

MySQL se refiere a una herramienta de administración de base de datos, así como también al tipo de base de datos que administra. Dichas bases de datos son utilizadas por aplicaciones tales como boletines de noticias y carros de compras de venta al por menor. Esta pantalla le permite crear, editar y eliminar bases de datos y cuentas de usuario de MySQL, y le da a los usuarios acceso a bases de datos específicas.

Solamente los usuarios de MySQL (diferentes de los usuarios de correo o de otro tipo) que tienen los privilegios de acceso correctos, pueden leer o escribir en la base de datos.

Crear una nueva base de datos de MySQL

  1. Ingrese un nombre de base de datos en la casilla Nueva base de datos.
  2. Haga clic en Crear bases de datos.

Para verificar la existencia de errores o corrupción en una base de datos de MySQL

  1. Seleccione una base de datos del menú Verificar base de datos.
  2. Haga clic en Verificar base de datos.

Para reparar una base de datos de MySQL con errores o corrupción de archivo

  1. Seleccione una base de datos del menú Reparar base de datos.
  2. Haga clic en Reparar base de datos.

Buscar una base de datos de MySQL

Si se administra una cantidad importante de bases de datos, se puede utilizar el campo Buscar arriba de la tabla Bases de datos actuales para localizar una base de datos en particular. Para buscar en la tabla Bases de datos actuales ingrese sus criterios en el campo Buscar y haga clic en Ir.

Eliminar una base de datos de MySQL

  1. Haga clic en Eliminar base de datos. Será dirigido a una nueva página.
  2. Haga clic en Sí para eliminar permanentemente la base de datos o en No para cancelar la eliminación.

Agregar un nuevo usuario de MySQL

Necesitará crear cuentas de usuarios de MySQL en forma separada a las cuentas de correo y administrador de red.

  1. Ingrese el nombre de usuario, contraseña y confirmación de la contraseña en las casillas apropiadas.
    Aviso: La casilla Seguridad le mostrará la seguridad de su contraseña. Si no puede imaginarse una contraseña segura, haga clic en el botón Generador de contraseñas para obtener una opción apropiada.
  2. Haga clic en Crear usuario.

Agregar un usuario de MySQL a la base de datos

Para que un usuario pueda ingresar a la base de datos, necesitará asignarlo a dicha base.

  1. Seleccione el usuario apropiado del menú Usuario.
  2. Seleccione la base de datos apropiada del menú Base de datos.
  3. Haga clic en Agregar.

Eliminar un usuario MySQL

  1. Haga clic en X. Será dirigido a una nueva página.
  2. Haga clic en Sí para eliminar permanentemente el usuario o en No para cancelar la remoción.

Eliminar un usuario de MySQL de una base de datos específica

  1. En la tabla Bases de datos actuales, localice al usuario cuyos privilegios desea eliminar.
  2. Haga clic en X. Será dirigido a una nueva página.
  3. Haga clic en Sí para eliminar permanentemente al usuario o en No para cancelar la eliminación.
