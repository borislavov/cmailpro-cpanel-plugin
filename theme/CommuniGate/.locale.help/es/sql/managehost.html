¿Qué es MySQL?

MySQL se refiere a una herramienta de administración de base de datos, así como también al tipo de base de datos que administra. Dichas bases de datos son utilizadas por aplicaciones tales como boletines de noticias y carros de compras de venta al por menor.

¿Que es la característica MySQL remota?

Las bases de datos en cPanel pueden ingresarse en forma remota, por otros servidores de red. Esto puede ser útil si desea permitir que las aplicaciones tales como carritos de compra o libro de invitados, en otros servidores ingresen a las bases de datos.

Esta pantalla le permite proporcionar a servidores externos, el ingreso a sus bases de datos MySQL.

Agregar un servidor de acceso

  1. Ingrese una dirección IP o nombre de dominio en la casilla Servidor.
    Nota: Se puede ingresar un nombre de dominio totalmente calificado (FQDN) o una dirección IP. También se puede utilizar el signo % como comodín. Por ejemplo, para permitir el ingreso desde todas las direcciones IP que comienzan con 208.77.188, puede ingresar 208.77.188.%.
  2. Haga clic en Agregar servidor.

Eliminar un servidor de acceso

  1. Haga clic en X Será dirigido a una nueva página.
  2. Haga clic en Sí para eliminar permanentemente el ingreso del servidor remoto o en No para cancelar la eliminación.
