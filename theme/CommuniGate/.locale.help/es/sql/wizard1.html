¿Qué es MySQL?

MySQL se refiere a una herramienta de administración de base de datos, así como también al tipo de base de datos que administra. Dichas bases de datos son utilizadas por aplicaciones tales como boletines de noticias y carros de compras de venta al por menor. El Asistente de MySQL le permite crear bases de datos y cuentas de usuario de MySQL.

Para ingresar nueva información en una base de datos, se puede utilizar el cliente phpMyAdmin instalado con cPanel.
