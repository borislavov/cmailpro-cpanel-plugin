Personalizar la interfaz de cPanel

El editor de personalización de cPanel le permite personalizar la interfaz de cPanel para sus usuarios.

  • Para duplicar un estilo de personalización, y de esa forma descargarlo y personalizarlo, haga clic en Crear o cargar personalización.
  • Para descargar un estilo de personalización, haga clic en Descargar o eliminar personalización.
  • Para cargar el estilo personalizado, haga clic en Crear o cargar personalización.
  • Para eliminar el estilo personalizado, haga clic en Descargar o eliminar personalización.

Aplicar y editar estilos instalados

La pantalla inicial del editor de personalización posee también una cantidad de opciones de estilos existentes en su servidor.

  • Para que un estilo quede disponible para los usuarios, haga clic en la casilla de selección Activado arriba del estilo.
  • Para utilizar un estilo para su propia cuenta de cPanel, haga clic en Aplicar a mi cuenta.
  • Para utilizar un estilo para todas las cuentas de usuario, haga clic en Aplicar a todas las cuentas.
  • Para seleccionar un estilo como tema predeterminado para las cuentas nuevas, haga clic en Definir como predeterminado para las cuentas nuevas.
  • Para editar un estilo utilizando cPanel, haga clic en Editar este estilo.
