Editar un estilo

Los estilos de personalización se pueden editar un mediante cPanel

Cuando se selecciona esta opción, se le presentan las siguientes opciones que le permiten editar diversas partes de la interfaz de cPanel:

  • Editar logotipos
  • Editar íconos
  • Editar encabezado/pie de página
  • Editar UI (interface de usuario)
  • Editar imágenes de vista previa
  • Editar páginas HTML
  • Editar hojas de estilo
