Modificar la imagen previa de un estilo

Se puede reemplazar una de las dos imágenes previas: la imagen grande o la imagen pequeña. (La imagen pequeña se muestra en la pantalla inicial del editor de personalización. La imagen grande se muestra cuando un usuario hace clic en la imagen pequeña en la pantalla inicial del editor de personalización).

1. Haga clic en Examinar.

  • Se recomienda que seleccione una captura de imagen del estilo, ya que ésto es lo más claro para sus usuarios, al momento de seleccionar una interfaz preferida.
  • Las imágenes de pantalla personalizadas deben tener los mismos nombres de archivo que los originales.
  • Se puede hacer clic en Restablecer para utilizar la imagen previa predeterminada.
  • Recuerde: Necesita efectuar este proceso tanto para las imágenes previas pequeñas como para las grandes.

2. Haga clic en Cargar imágenes en la parte inferior de la pantalla.

  • Se puede hacer clic en Restablecer todas las vistas previas para devolver ambas imágenes previas a sus valores predeterminados.
