Editar un archivo CSS de estilo (en inglés: cascading style sheet, (hojas en estilo de cascada))

1. Realice cualquier cambio o agregados en la casilla del editor de archivos CSS

2. Haga clic en Guardar CSS para guardar los cambios.

  • Se puede hacer clic en Revertir para devolver la hoja de estilos a sus valores predeterminados.

Advertencia: Si no está utilizando el sistema de signos UTF-8, debe tener cuidado especial para no utilizarlo al crear sus páginas CSS. En este caso, utilizar datos del sistema de signos UTF-8 corromperá la página.
