Editar la hoja de estilo HTML

Existen dos opciones para editar un archivo de estilo de HTML directamente: puede utilizar el editor de HTML de cPanel, o cargar su propio archivo *.html personalizado.

Utilizar el editor de HTML de cPanel

1. Haga clic en Editar para editar el HTML en el editor.

  • Se puede hacer clic en Revertir para deshacer los cambios realizados en el editor.

2. Haga clic en Guardar para almacenar los cambios realizados con el editor.

  • Se puede hacer clic en Restablecer para devolver la página HTML al valor predeterminado.
  • Se puede hacer clic con el botón derecho en el nombre de archivo HTML para descargarlo a su computadora, y así poder editarlo mediante un editor de texto.

Cargar su página HTML personalizada

1. Haga clic en Examinar, junto al archivo que desea reemplazar, y navegue hasta el archivo *.html que desea cargar.

2. Haga clic en Cargar imágenes en la parte inferior de la pantalla.

  • Se puede hacer clic en Restablecer para devolver la página HTML al valor predeterminado.
  • Se puede hacer clic con el botón derecho en el nombre de archivo HTML para descargarlo a su computadora, y así poder editarlo mediante un editor de texto.

Advertencia: Si no está utilizando el sistema de signos UTF-8, debe tener un cuidado especial para no utilizarlo al crear sus páginas HTML. En este caso, utilizar datos del sistema de signos UTF-8 corromperá la página.
